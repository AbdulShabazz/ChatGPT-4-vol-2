## OpenAI ChatGPT v1.2025.330 Windows.Shortcut.Keys.md

**Q: What are all of the latest Windows (icon) shortcut keys (e.g., Win + D)?**

**A:** Here is a consolidated list of Windows-logo (Win) shortcuts in Windows 11, as documented by Microsoft plus recent additions, as of December 2025.([Microsoft Support][1])

I’ve grouped them so they’re easier to scan.

---

### Core system, Start, Settings, search

* **Win** – Open/close Start.
* **Win + A** – Open Quick Settings / action center (Wi-Fi, volume, brightness, etc.).
* **Win + X** – Open Quick Link menu (same as right-click Start).
* **Win + I** – Open Settings.
* **Win + U** – Open Settings directly to Accessibility.
* **Win + L** – Lock the PC.
* **Win + Q** – Open search.
* **Win + S** – Open search.
* **Win + C**

  * When Copilot is available: Open Microsoft Copilot / Microsoft 365 Copilot.
  * When Copilot is disabled/not available: Open/close Windows search.([Microsoft Support][1])
* **Win + J** – Open Windows Recall (only on supported hardware / builds).([Microsoft Support][1])
* **Win + N** – Open Notification Center + calendar.
* **Win + W** – Open Widgets.
* **Win + B** – Move focus to first icon in the notification area.
* **Win + Pause** – Open System > About page in Settings.
* **Win + O** – Lock device orientation.
* **Win + K** – Open Cast (wireless display and audio) from Quick Settings.
* **Win + F** – Open Feedback Hub.

---

### File Explorer, Run, general launching

* **Win + E** – Open File Explorer.([Microsoft Support][1])
* **Win + R** – Open Run dialog.
* **Win + G** – Open Xbox Game Bar.
* **Win + T** – Cycle forward through apps on taskbar (all monitors).
* **Win + Shift + T** – Cycle backward through apps on taskbar.

---

### Window and desktop management (snap, minimize, restore)

* **Win + D** – Show/hide desktop.
* **Win + M** – Minimize all windows.
* **Win + Shift + M** – Restore minimized windows.
* **Win + Home** – Minimize/restore all windows except the active one.
* **Win + Up Arrow** – Maximize active window.
* **Win + Down Arrow** – Minimize active window (or restore from maximized).
* **Win + Left Arrow** – Snap active window to left half.
* **Win + Right Arrow** – Snap active window to right half.
* **Win + Alt + Up Arrow** – Snap active window to top half of screen.
* **Win + Alt + Down Arrow** – Snap active window to bottom half.([Microsoft Support][1])
* **Win + Shift + Up Arrow** – Stretch window vertically (top and bottom of screen).
* **Win + Shift + Down Arrow** – Restore a maximized/snapped window.
* **Win + , (comma)** – Peek at desktop while held.
* **Win + Z** – Open Snap Layouts.
* **Win + Tab** – Open Task View (virtual desktops and timeline).

---

### Virtual desktops

* **Win + Tab** – Open Task View.
* **Win + Ctrl + D** – Create a new virtual desktop.
* **Win + Ctrl + Right Arrow** – Switch to virtual desktop on the right.
* **Win + Ctrl + Left Arrow** – Switch to virtual desktop on the left.
* **Win + Ctrl + F4** – Close current virtual desktop.([Microsoft Support][1])

---

### Taskbar and pinned-app shortcuts (Win + number)

All “number” keys are positions on the taskbar, counting from left starting at 1:

* **Win + number (0–9)** – Open/switch to pinned app at that position.
* **Win + Shift + number (0–9)** – Open new instance of pinned app.
* **Win + Ctrl + number (0–9)** – Switch to last active window of pinned app.
* **Win + Ctrl + Shift + number (0–9)** – Open a new instance of pinned app as administrator.
* **Win + Alt + number (0–9)** – Open Jump List of the pinned app at that position.
* **Win + Alt + Enter** (with a taskbar item focused) – Open taskbar settings.([Microsoft Support][1])

---

### Input language, IME, keyboard layout

* **Win + Spacebar** – Cycle forward through input languages/keyboard layouts.
* **Win + Shift + Spacebar** – Cycle backward through input languages/keyboard layouts.
* **Win + Ctrl + Spacebar** – Switch to previously selected input method.
* **Win + / (forward slash)** – Start IME reconversion (IME-specific).([Microsoft Support][1])

---

### Clipboard, emoji, text and dictation

* **Win + V** – Open clipboard history (must be enabled first).
* **Win + . (period)** – Open emoji panel (also symbols, kaomoji).
* **Win + ; (semicolon)** – Same as above (emoji panel).
* **Win + H** – Open voice typing / dictation.
* **Win + Alt + H** – Move keyboard focus to voice typing dialog.([Microsoft Support][2])

---

### Capture (screenshots, recording)

* **Win + PrtScn** – Capture full screen and save to Pictures\Screenshots.
* **Win + Shift + S** – Open snipping overlay to capture a region to clipboard (Snipping Tool).
* **Win + Shift + R** – Select a region to record as a screen recording (saved as MP4 in Videos\Screen Recordings; opens in Snipping Tool).([Microsoft Support][1])

---

### Magnifier and display (zoom, HDR)

* **Win + + (plus)** – Turn on Magnifier and zoom in.([Microsoft Support][2])
* **Win + – (minus)** – Zoom out in Magnifier.
* **Win + + / – while Magnifier is on** – Zoom in/out.
* **Win + Esc** – Turn off Magnifier.
* **Win + Ctrl + M** – Open Magnifier settings.
* **Win + Alt + B** – Toggle HDR on/off (where supported).([Microsoft Support][1])

---

### Narrator, speech, accessibility tools

* **Win + Ctrl + Enter** – Start Narrator.
* **Win + Ctrl + N** – Open Narrator settings.
* **Win + Ctrl + S** – Turn on Windows Speech Recognition.
* **Win + Ctrl + O** – Turn on the On-Screen Keyboard.
* **Win + A** – Open Quick Settings (often used for accessibility toggles).
* **Win + U** – Open Accessibility section in Settings (duplicate with above group for completeness).([Microsoft Support][2])

---

### Audio, mic, sound output

* **Win + Ctrl + V** – Open sound output quick settings page (output device, spatial sound, volume mixer).
* **Win + Alt + K** – Mute/unmute microphone in supported apps.([Microsoft Support][1])

---

### Network, assistance, devices

* **Win + Ctrl + F** – Search for devices on a network.
* **Win + Ctrl + Q** – Open Quick Assist (remote assistance).
* **Win + K** – Open Cast (wireless displays, etc.).([Microsoft Support][1])

---

### Graphics / troubleshooting

* **Win + Ctrl + Shift + B** – Wake the device / reset graphics driver when the screen is blank or black (beeps and flickers briefly).([Microsoft Support][1])

---

### Punctuation – new dashes (recent Windows 11 builds)

These are rolling out in newer Windows 11 releases:

* **Win + – (minus)** – Insert an en dash (–).
* **Win + Shift + – (minus)** – Insert an em dash (—).([TechRadar][3])

Note: these may conflict with Magnifier zoom shortcuts depending on configuration; behavior can depend on your build and settings.
