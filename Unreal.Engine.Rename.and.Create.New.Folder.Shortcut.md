## OpenAI ChatGPT v1.2025.330 Create.New.Folder.Shortcut.md

**Q: Is there a shortcut to create a new folder in unreal engine content browser?**

**A:** Yes.

By default in the Unreal Editor Content Browser:

* New folder: **Ctrl + Shift + N** (Windows)
* New folder: **Cmd + Shift + N** (macOS) ([logickeyboard.com][1])

You can confirm or remap this under:
**Edit → Editor Preferences → Keyboard Shortcuts → Content Browser → Create Folder**.
```
[1]: https://logickeyboard.com/shortcut/UNREAL220812.pdf?utm_source=chatgpt.com "Unreal Engine"
```
**Q: How do you rename the folder?**

**A:** In the Unreal Engine Content Browser:

* Select the folder and press **F2** to rename.
* Or **click once to select**, then **click the name again** (slow double-click on the label) to enter rename mode.

This works for both assets and folders.
