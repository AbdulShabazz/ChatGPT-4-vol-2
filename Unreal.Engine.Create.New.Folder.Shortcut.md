## OpenAI ChatGPT v1.2025.330 Create.New.Folder.Shortcut.md

**Q: Is there a shortcut to create a new folder in unreal engine content browser?**

**A:** In the Unreal Engine Content Browser:

* Select the folder and press **F2** to rename.
* Or **click once to select**, then **click the name again** (slow double-click on the label) to enter rename mode.

This works for both assets and folders.
